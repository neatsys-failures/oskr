Detailed Description

template<typename Transport>
struct oscar::TransportMeta< Transport >

A struct required by C++, so Transport's subclass can define essentials ahead of instantiating base class.

Each specialization must include:

  • Associated type Address. It is assumed to be suitable for value semantic.
  • Compile-time constant std::size_t expression BUFFER_SIZE, the maximum length of serialized tx message. Sender's message length is bounded by this so transport do not need to provide dynamic allocated buffer.
